# Secure Payment Gateway System

A secure payment gateway prototype designed for e-commerce applications. The project demonstrates secure payment processing, authentication, and backend security while following modern cybersecurity practices.

## Features

- User registration and login
- Secure payment processing
- PayPal payment integration
- JWT-based authentication and authorization
- Password hashing with bcrypt
- HTTPS/TLS secure communication
- SQL Injection protection
- Rate limiting
- PayPal webhook signature verification
- Security testing and validation

## Technologies

- Node.js
- Express.js
- JavaScript
- MongoDB
- JWT
- bcrypt
- PayPal REST API
- HTTPS / TLS

## Security Features

- Secure password hashing
- Token-based authentication
- Access control
- Input validation
- SQL Injection mitigation
- Rate limiting
- Webhook verification
- TLS encryption

## Testing

- SQL Injection testing
- Authentication validation
- JWT authorization testing
- Rate limiting verification
- HTTPS/TLS validation
- PayPal webhook forgery testing
